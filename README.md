# Template Project
This is a skeleton project from which real projects can be started.

It supports:
* Test-driven development using `unittest`
* Packaging using `setuptools`
* Caching of test files in a local `data/` folder
* A `tmp/` folder where most everything (except for packaging, see below) is ignored.

# Testing
Run `python3 package_test.py` from the root folder.
In VS Code, see: https://code.visualstudio.com/docs/python/testing

# Packaging
Run `python3 -m build` from `tmp/packaging/`.
For more information, see https://packaging.python.org/en/latest/tutorials/packaging-projects/
